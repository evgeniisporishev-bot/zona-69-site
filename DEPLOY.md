# Публикация сайта ZONA 6/9 бесплатно

Сайт использует Node.js сервер, потому что форма заявки отправляет уведомления в Telegram.

## Бесплатный вариант: Render

Render подходит для этого сайта, потому что на бесплатном тарифе можно запустить Node.js Web Service.

Важный нюанс бесплатного тарифа: если 15 минут никто не заходит на сайт, сервис засыпает. Первый следующий заход может открываться около минуты.

## Что нельзя публиковать

Не загружайте файл `.env`. В нем лежат секретные данные Telegram-бота.

Эти значения нужно добавить в настройках Render:

```text
TELEGRAM_BOT_TOKEN=ваш_токен_бота
TELEGRAM_CHAT_ID=ваш_chat_id
```

## Как опубликовать

1. Создайте аккаунт на GitHub.
2. Создайте новый репозиторий, например `zona-69-site`.
3. Загрузите файлы сайта в репозиторий.
4. Создайте аккаунт на Render.
5. В Render нажмите `New` -> `Web Service`.
6. Подключите репозиторий `zona-69-site`.
7. Выберите Free plan.
8. Проверьте настройки:
   - Build command: `npm install`
   - Start command: `npm start`
9. В Environment добавьте:
   - `TELEGRAM_BOT_TOKEN`
   - `TELEGRAM_CHAT_ID`
10. Нажмите Deploy.

## После публикации

Откройте публичную ссылку Render и отправьте тестовую заявку. Если все настроено правильно, сообщение придет в Telegram.
